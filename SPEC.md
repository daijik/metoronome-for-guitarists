# 仕様書 — Metronome for Guitarists

## 概要

ギタリストの練習セッションをサポートするウェブメトロノーム。  
BPM・拍子を設定してメトロノームを鳴らしながら、あらかじめ決めた練習時間をカウントダウンできる。

---

## 機能一覧

### 1. メトロノーム

| 項目 | 仕様 |
|------|------|
| BPM 範囲 | 30〜240 |
| BPM 変更方法 | スライダー / 数値直接入力 / ±ボタン（1刻み） |
| 拍子 | 2/4, 3/4, 4/4, 6/8 |
| 音源 | Web Audio API（オシレーター）。サーバー不要 |
| 1拍目 | 1000 Hz, 音量 0.8（アクセント） |
| 2拍目以降 | 800 Hz, 音量 0.5 |
| タイミング精度 | Lookahead scheduler（25ms 間隔でスケジューリング、100ms 先読み）。`setInterval` のドリフトを回避 |

### 2. ビート表示

- 拍数ぶんのドットを横並びに表示
- 現在の拍に対応するドットが点灯（青）
- 1拍目のドットはオレンジ（アクセント色）で点灯

### 3. 練習タイマー

| 項目 | 仕様 |
|------|------|
| 設定方法 | プリセットボタン（1/3/5/10/15/30分）または数値直接入力（分単位） |
| 最小値 | 1分（0分以上を受け付けるが実用上1分〜） |
| 表示形式 | MM:SS |
| プログレスバー | 残り時間 / 設定時間 の割合で横幅が変化 |
| 残り10秒以下 | タイマー表示が赤くフラッシュ |
| 設定変更 | 再生中はタイマー設定を変更不可（UI を disabled にする） |

### 4. Start / Stop

| 操作 | 動作 |
|------|------|
| Start | メトロノーム開始、タイマーを設定値から開始してカウントダウン |
| Stop（ボタン） | メトロノーム停止、タイマーを設定値にリセット |
| タイマーが 0 到達 | メトロノーム自動停止、タイマーを設定値にリセット |

## 既知の制限

- タイマーは 1 秒間隔の `setInterval` で更新するため、実際の経過時間との誤差が最大 ±1 秒程度生じる
- ブラウザのタブを非アクティブにすると `setInterval` が間引かれることがあり、タイマーが若干遅れる場合がある
